# 💻 Problemas de Algoritmos e Estruturas de Dados em Python (Neps Academy)

## 🎯 Sobre o Repositório

Este repositório é um portfólio técnico contínuo dedicado à resolução de desafios de **Lógica de Programação, Algoritmos e Estruturas de Dados** do **Neps Academy**.

O objetivo aqui vai muito além de simplesmente obter um status de "Accepted" nos compiladores. O foco principal é a aplicação real dos fundamentos da Engenharia de Software, priorizando:

* **Clean Code (Código Limpo):** Soluções legíveis, modulares e focadas em redução de carga cognitiva.
* **Otimização:** Uso inteligente dos recursos nativos do Python e análise rigorosa de formatação.
* **Storytelling Técnico:** Documentação clara das decisões arquitetônicas (trade-offs) por trás de cada linha de código.
